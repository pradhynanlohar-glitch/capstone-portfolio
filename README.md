# Front-End Developer Capstone - Portfolio

This repository contains a simple single-page portfolio for the IBM Front-End Developer Capstone assignment.  
Includes: About, Skills, Projects, Recommendations sections and a small JS module to add recommendations.

## Files
- `index.html` — single-file portfolio (or split HTML/CSS/JS)
- `simple-interest.sh` — bash script for simple interest calculation
- `LICENSE` — Apache 2.0
- `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`

## How to run
Open `index.html` in any modern browser (Chrome/Edge) to view the portfolio UI and test features.
